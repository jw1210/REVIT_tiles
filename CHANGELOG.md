# CHANGELOG - AntiGravity 磁磚計畫

本文件記錄了 AntiGravity 磁磚計畫外掛的開發歷程與重大修正內容。

## 🎯 V3.9 版本亮點 — 自訂轉角間距版

**版本 V3.9 (2026-03-15)** 廢除自動抓取預設灰縫機制，改為即時彈窗輸入：

- ✨ **自訂轉角灰縫 (Custom Corner Gap)** — 選完收邊形式後立刻彈出輸入視窗，預設 1mm，支援精確到小數點的參數控制。
- ✨ **動態 WPF 輸入介面** — 完全基於 C# 動態生成，免除 XAML 檔案依賴，部署更簡潔。
- ✨ **操作流暢度提升** — 支援 Enter 鍵確認输入並直接進入框選模式，實現「即時設定、即時切割」。

👉 **詳細技術文檔**: 見 [V39_TECHNICAL_GUIDE.md](V39_TECHNICAL_GUIDE.md)

## 🎯 V3.8 版本亮點 — 核心補強與 UX 優化版

**版本 V3.8 (2026-03-15)** 專注於操作回饋的直覺性與底層交易處理的強韌度：

- ✨ **明確選取提示 (Clear UX Prompts)** — 批次分割與手動收邊均加入「Enter 鍵確認」提示，解決使用者選完後不知如何觸發下一步的困惑。
- ✨ **TransactionGroup 交易保護模式** — 導入 `TransactionGroup` 架構，將合併、切割等複數步驟封裝。確保任何幾何運算失敗時能「零殘留」回退，並彈出原因視窗。
- ✨ **幾何防呆補強** — 修正局部轉角在特定角度下可能產生的靜默失敗。

👉 **詳細技術文檔**: 見 [V38_TECHNICAL_GUIDE.md](V38_TECHNICAL_GUIDE.md)

## 🎯 V3.7 版本亮點 — 批次與幾何大師版

**版本 V3.7 (2026-03-15)** 引入了多目標批次處理與跨維度的轉角收邊演算法：

- ✨ **批次多選分割 (Batch Processing)** — 選取指令後可一次框選多面牆或樓板，顯著提升大面積排磚效率。
- ✨ **三維全向度轉角 (3D Corner Join)** — 突破平面限制，支援牆與牆、牆與地、地與地的 45 度切角或蓋磚收邊。
- ✨ **UI 操作直覺化** — Ribbon 面板新增獨立的「局部轉角」控制按鈕。

## 🎯 V3.6 版本亮點 — 全域網格控制版

**版本 V3.6 (2026-03-15)** 徹底解決了網格線在「已隱藏」狀態下無法被重新找回的邏輯死結：

- ✨ **全域網格撈取 (Global Grid Retrieval)** — 搜尋範圍從「當前視圖」擴大至「整個專案文件 (doc)」。確保即使網格線處於隱藏狀態，系統仍能精準捕捉並進行開關切換。
- ✨ **狀態感知邏輯優化** — 自動判斷視圖中是否存在任何可見網格，實施「一現則全隱，全隱則全現」的直覺操作邏輯。

## 🎯 V3.5 版本亮點 — 幾何對位加強版

**版本 V3.5 (2026-03-15)** 攻克了門窗開口對交丁排的干擾，並強化了視圖控制的穩定性：

- ✨ **交丁高度群組化 (Precision Height Grouping)** — 核心邏輯升級！自動偵測並群組位於同一水平高度的碎塊零件，確保交丁排 (Running Bond) 在跨越門窗後依然保持完美的 A-B 對位。
- ✨ **強制網格開關 (Forced Grid Toggle)** — 徹底解決參考平面在特定視圖下無法隱藏的 Bug，改用直接元素控制 (HideElements) 實現 100% 可靠的視覺切換。

## 🎯 V3.4 版本亮點 — UX 邏輯加固版

**版本 V3.4 (2026-03-14)** 掃除了介面交互中的「邏輯地雷」，大幅提升系統穩定性：

- ✨ **輸入防護網 (Input Hardening)** — 全面採用 `double.TryParse` 攔截格式錯誤，杜絕因格式錯誤導致的 DLL 崩潰。

- ✨ **動態引導介面 (Dynamic Context)** — 優化交丁排面板的連動顯示邏輯，引導使用者專注於當前選定的排列式樣指標。

## [V3.8.0] - 2026-03-15 (核心補強)

### Added
- **UX 提示語**：在 `PickObjects` 指令中加入明確的 `(選取完成請按 Enter 鍵確認)`。
- **強韌錯誤回報**：捨棄靜默 catch，若指令失敗一律彈出 `TaskDialog` 告知幾何原因。

### Changed
- **交易架構**：從單一 `Transaction` 改為 `TransactionGroup`。將幾何運算、零件合併、三維切割分段處裡。

### [V3.7.0] - 2026-03-15 (批次與幾何)

### Added

- **批次多選優化**：`ManualCornerJoinCommand` 全面支援 `PickObjects` 框選與 Enter 鍵確認流程。
- **手動收邊 (Final)**：新增 45 度切角 (Miter) 與 蓋磚 (Cover) 的「鎔鑄多切」模式。

### Changed

- **核心演算法 (Robust v2)**：導入「法向量最大化交叉比對」機制，徹底解決幾近平行面的判斷錯誤問題。
- **切割線延伸**：將幾何切割線大幅延伸，確保整排框選的零件能一次切斷。

## [V3.6.0] - 2026-03-15 (全域網格控制)

### Fixed

- **網格開關邏輯修復**：解決當網格完全隱藏時，`FilteredElementCollector(doc, activeView.Id)` 無法找到對象導致開關失效的問題。

### Changed

- **搜尋策略**：將 `FilteredElementCollector` 作用域提升至 `Document` 層級，跨視圖生命週期管理網格狀態。

## [V3.5.0] - 2026-03-15 (幾何對位加強)

### Fixed

- **交丁修復**：修正門窗截斷導致的 Alternating Pattern (A/B) 錯位問題。
- **網格開關修復**：修正 `ToggleGridCommand` 無法在所有情況下隱藏綠線的 Bug。

### Changed

- **垂直分割策略**：在 `PerformDivision` 中導入 `Tolerance` (0.5 CellHeight) 高度判斷機制。

## [V3.4.0] - 2026-03-14 (UX 邏輯加固)

- **地雷邏輯修復**：修正非數字輸入可能導致的 Parse Error。

- **快速鍵綁定**：確保全視窗範圍內的 Enter/Esc 鍵盤映射。

## [V3.3.0] - 2026-03-14 (UI 交互優化)

- **UI 佈局更新**：在磁磚尺寸與灰縫設定區塊分別加入對應的對調按鈕。
