# TilePlanner V3.7 技術指南 — 批次效能與三維幾何收邊

## 1. 批次處理機制 (Batch Selection)

在 V3.7 中，我們將磁磚生成邏輯從單選升級為「批次選取」：

- **API 變更**：從 `Selection.PickObject` 升級為 `Selection.PickObjects`。
- **操作流程**：選取指令後，您可以拉框或按住 Ctrl 點選多個牆面/樓板，最後按下 Revit 選項列的「完成 (Finish)」即可觸發批次切割。
- **錯誤隔離**：內部採用 `foreach` 封裝，若其中一面牆因幾何過於複雜失敗，系統會自動跳過並繼續處理下一面，不會中斷整個任務。

## 2. 三維全向度轉角演算法 (3D Corner Join Algorithm)

這是 V3.7 最核心的突破，解決了「非垂直牆面」或「牆與樓板接合」的收邊難題。

### 向量座標運算流程

1.  **法向交叉乘積**：取得兩塊磁磚的最大表面法向量 $n_A$ 與 $n_B$，計算其外積 $axis = n_A \times n_B$。此 Axis 即為轉角接縫線的方向。
2.  **動態草圖平面**：以該 Axis 為法向，建立一個臨時的 2D 草圖平面 (SketchPlane)。
3.  **三面相交解算**：
    - 將 A 磚表面、B 磚表面、以及上述草圖平面進行「三平面相交運算」。
    - 解算出 4 個關鍵交點，定義出轉角空間。
4.  **幾何投影切割**：
    - **45度切角 (Miter)**：連接空間對角線。
    - **蓋磚 (Cover)**：沿著「被蓋磚」的內側邊緣線進行垂直切割。

### 支援場景

- **陽角/陰角**：牆壁轉歸處。
- **女兒牆頂蓋**：牆壁與頂部水平蓋磚。
- **樓板與牆踢腳**：水平面與垂直面的交接。

## 3. 部署注意事項 (Deployment)

- **指針更新**：本版本繼續採用 `.addin` 指向本地 DLL 的指針模式，避免 Revit 運行期間鎖定 AppData 檔案。
- **指令登錄**：請確認 `ManualCornerJoinCommand` 已在清單中註冊。
