# Master CHANGELOG - AntiGravity 磁磚計畫 (TilePlanner)

本文件為 AntiGravity 磁磚計畫外掛的 **唯一權威版本紀錄 (Master Version History)**。所有歷次的開發細節、技術變革、介面優化以及錯誤修正均整合於此，往後版本更新亦將接續紀錄。

## 🎯 V3.9.2 — 終極完美版 (Ultimate Perfect Version)
**日期**: 2026-03-15 | **目標**: 體驗極致化與靜默操作

本版本為 V3.x 系列的巔峰之作，徹底解決了操作卡頓感與 Revit 原生警告彈窗的干擾：

- 🚀 **Enter 鍵強效攔截 (`PreviewKeyDown`)**：
    - 升級 TextBox 監聽器，確保在間距對話框中按下 Enter 鍵立即執行「確定」，無須滑鼠點擊。
    - 解決了傳統 `IsDefault` 在 Revit 環境下失效的問題。
- 🤫 **終極靜默操作 (`ResolvedErrorSwallower`)**：
    - 深入掛載 `LocalWarningSwallower`，除了自動刪除警告外，新增 **「錯誤自動化解 (Error Resolver)」** 邏輯。
    - 100% 杜絕所有 Revit HTML 格式的「零件將被刪除」警告，實現霸道式的靜默執行。
- 🌐 **全域幾何核心同步**：
    - 完整融合 V3.9.1 的全選全能偵測邏輯，支援牆面、壓頂、地板全場景。

## 🎯 V3.9.1 — 全域軸心偵測版 (Universal Axis Detection)
**日期**: 2026-03-15 | **目標**: 突破垂直面限制

解決了「壓頂磚 (Wall Caps)」與「地磚」等水平面收邊時的幾何偏移問題：

- ✨ **全域軸心偵測**：自動判定磁磚平面（立面、平面或斜面），動態調整轉角軸心向量。
- ✨ **側面幾何精確導向 (`GetSideFaces`)**：
    - 捨棄模糊的「抓取最大面」邏輯，改用向量內積精確過濾出垂直於軸心的側面作為投影基準。
- 🛠️ **Transaction 模型優化**：修正 `TransactionGroup` 在特定環境下的編譯報錯，改將 `FailureHandlingOptions` 直接綁定於內部交易。

## 🎯 V3.9 — 自訂轉角間距版
**日期**: 2026-03-15 | **目標**: 高度靈活性界面

廢除自動抓取灰縫硬編碼，賦予使用者完全的控制權：

- ✨ **動態 WPF 輸入介面 (No-XAML)**：
    - 完全由 C# 動態生成 UI 視窗，零外部檔案依賴，部署極簡化。
- ✨ **即時自訂間距**：選取形式後即刻彈窗輸入 (預設 1mm)，支援精確小數點參數。
- ⚡ **操作流暢優化**：視窗開啟即自動全選數值，支援鍵盤盲打。

## 🎯 V3.8 — 核心補強與 UX 優化版
**日期**: 2026-03-15 | **目標**: 交易保護與直覺引導

- ✨ **明確 UX 提示**：選取指令中加入 `(選後按 Enter 確認)`，解決選完後不知如何觸發的困惑。
- 🛡️ **TransactionGroup 保護架構**：
    - 將合併、刷新、切割封裝於單一容器中，確保任何幾何失敗都能「零殘留」回退 (Rollback)。
- 📢 **報錯主動告知**：捨棄靜默 catch，若不符切割條件會主動彈窗告知原因。

## 🎯 V3.7 — 批次與幾何大師版
**日期**: 2026-03-15 | **目標**: 效率提升與三平面解算

- 🌊 **批次多選分割**：從單選升級為 `PickObjects` 模式，支援一次框選多面牆/多個室內地板。
- 📐 **三維幾何收邊演算法**：
    - 實作「牆與牆、牆與地、地與地」的全向度 45 度切角。
    - 利用「三面相交解算」精準找出內角 (I) 與外角 (O) 空間點。
- 📏 **切割延伸 (50ft)**：確保超大型零件能在投影面上一刀切斷。
