# TilePlanner V2.4 — 快速部署與測試指南

## 📦 部署步驟（開發環境）

### Step 1: 編譯專案
```powershell
cd c:\Users\jenwe\Desktop\REVIT_tiles
msbuild TilePlanner.sln /p:Configuration=Release /p:Platform=x64
```

### Step 2: 複製至 Revit 外掛目錄
```powershell
$addinsPath = "$($env:AppData)\Autodesk\Revit\Addins\2025"
Copy-Item "TilePlanner\bin\Release\TilePlanner.dll" -Destination $addinsPath
Copy-Item "TilePlanner.addin" -Destination $addinsPath
```

### Step 3: 啟動 Revit 2025
- 開啟 Revit
- 在 Ribbon 中應該看到「**磁磚計畫**」標籤
- 包含 4 個按鈕：
  1. 建立磁磚計畫
  2. 移除磁磚計畫
  3. 整體連動
  4. **顯示/隱藏網格** ← V2.4 新增

## 🧪 快速驗證測試 (5 分鐘)

### 測試場景 A: 正排網格建立

1. **準備**:
   - Revit 新建空專案
   - 建立一面牆：寬 5000 mm、高 3000 mm、厚 200 mm

2. **執行**:
   - 選取牆體 → 右鍵 → 建立零件 (Create Parts)
   - 等待完成（視圖變灰色表示進行中）
   - 點擊「**建立磁磚計畫**」
   - 選取剛剛建立的零件
   - 對話框：預設值（200×200 mm，灰縫 3 mm，正排）
   - 按「**確定**」

3. **預期結果**:
   - ✓ 視圖自動轉為「展示零件」
   - ✓ 3D 視圖中可見灰縫凹陷
   - ✓ 完成對話框顯示：正排、200×200、灰縫 3×3
   - ✓ **沒有任何約束衝突警告**

4. **驗證網格鎖定**:
   - 點擊「**顯示/隱藏網格**」→ 綠色網格出現
   - 用 AL (Align) 工具拖動任一條網格線
   - ✓ **整個網格應同步平移**
   - ✓ 磁磚不變形

### 測試場景 C: 交丁排模式

1. **準備**:
   - 新建另一面牆（保持 5000×3000 mm）

2. **對話框設定**:

3. **預期結果**:
   - ✓ 奇數行應向右偏移 50%（磚寬的一半）
   - ✓ 視覺上應呈「之字形」排列
   - ✓ 完成訊息應顯示「交丁排」

## 📋 故障排除速查表

### 問題 1: 「建立磁磚計畫」按鈕無反應
**解決**:
- 確認已點擊「建立零件」
- 選取的物件應是「零件」(Part)，而非原始牆體
- 檢查 `PartSelectionFilter` 的邏輯

### 問題 2: 執行完成後出現「約束不滿足」錯誤
**解決** (V2.4 不應發生):
- 按 Ctrl+Z 復原
- 檢查 `ClearOldGridElements()` 是否被呼叫
- 查看 Revit 日誌檔案: `%Temp%\Revit_*.log`

### 問題 3: 灰縫凹陷無法在 3D 顯示
**解決**:
- 確保視圖「零件可見性」= 「展示零件 (Show Parts)」
- 確認 PartMaker 的「分割間隙」參數已被設定
- 嘗試 Regenerate (F9)

### 問題 4: 網格無法用 AL 工具平移
**解決**:
- 檢查網格是否與牆壁邊界的參照平面混在一起
- 確認 `ReferenceArray` 只包含「TileGrid_」開頭的平面
- 檢查連續標註的鎖定狀態 (IsLocked = true)
