# 已知問題與技術限制 (Known Issues & Technical Limitations)

以下記錄 TilePlanner 幾何引擎目前已知的技術細節、運算補償與待優化項目。

## [幾何補償] 1.52cm (0.05ft) 廢料延伸
- **描述**：為確保轉角斜切 (Miter Join) 的 Boolean 布林切割絕對成功，核心引擎在執行時會強制進行 0.05 呎 (1.52cm) 的物理重疊。
- **影響**：這會導致被排除的「廢料零件」在視覺上會比實際牆面外表面長出 1.52cm。
- **狀態**：此為預期中的穩定性補償，暫不移除以維持切割成功率。

## [待解問題] 轉角延伸尺寸落差 (Miter Extension Size Discrepancy)
- **描述**：目前的「斜切接合 (Miter Join)」採用造型控點延伸方式，這會使位於轉角處的第一塊磁磚外部尺寸（含延伸量）大於預先設定的規格（例如 20cm）。
- **流程建議**：建議先執行「轉角接合」，再進行「磁磚分割」以獲得最完美的對位成果。
- **未來計畫**：考慮在未來版本中加入「起點位移補償」或「接合後二次對位」機制。

## [待解問題] 性質欄參數不顯示 (Missing Properties on Part)
- **描述**：V4.5 系列導入了性質欄驅動功能，但在特定 Revit 環境或舊有文件中，`Tile_HGroutGap` 等參數可能無法出現在零件 (Parts) 的性質面板。
- **原因分析**：初步判定為 Revit 零件類別對動態綁定參數 (Project Parameters) 的顯示限制，或是 GUID 衝突。
- **狀態**：暫時記錄為已知問題，V4.5.0 功能保留但顯示可能受限。
