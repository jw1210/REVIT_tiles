# TilePlanner V3.7 技術指南 — 批次效能與三維幾何收邊

## 1. 批次處理機制 (Batch Selection)

在 V3.7 中，我們將磁磚生成邏輯從單選升級為「批次選取」：

- **API 變更**：從 `Selection.PickObject` 升級為 `Selection.PickObjects`。
- **操作流程**：選取指令後，您可以拉框或按住 Ctrl 點選多個牆面/樓板，最後按下 Revit 選項列的「完成 (Finish)」即可觸發批次切割。
- **多選確認流**：手動收邊指令現在支援「框選兩側零件」後按 **Enter** 確認，大幅提升大量零件收邊速度。
- **錯誤隔離**：內部採用 `foreach` 封裝，若其中一面牆因幾何過於複雜失敗，系統會自動跳過並繼續處理下一面，不會中斷整個任務。

## 2. 三維全方位轉角演算法 (Improved 3D Corner Join)

這是 V3.7 最核心的突破，解決了「非垂直牆面」或「牆與樓板接合」的收邊難題。

### 向量座標運算流程

1. **智慧交叉比對 (Robust Cross Product)**：遍歷兩側磁磚的所有表面，尋找法向量外積最大的兩面。這能確保在選取到「側面」或「微斜面」時，系統仍能精準找出真正的主選取投影面。
2. **動態草圖平面**：以計算出的 Axis ($n_A \times n_B$) 為法向，建立一個臨時的 2D 草圖平面 (SketchPlane)。
3. **三面相交解算**：
   - 將 A 磚表面、B 磚表面、以及上述草圖平面進行「三平面相交運算」。
   - 解算出 4 個關鍵交點，定義出轉角空間，並自動區分內角 (I) 與外角 (O)。
4. **幾何投影切割**：
   - **45度切角 (Miter)**：連接空間對角線。
   - **蓋磚 (Cover)**：沿著「保持完整側」的表面法向進行垂直切割。
   - **延伸切割**：切割線已大幅延伸 (50ft)，確保拉框選取的多塊零件能一次全部切斷。

### 支援場景

- **陽角/陰角**：牆壁轉角處。
- **女兒牆頂蓋**：牆壁與頂部水平蓋磚。
- **樓板與牆踢腳**：水平面與垂直面的交接。
