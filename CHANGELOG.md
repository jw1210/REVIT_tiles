# CHANGELOG - AntiGravity 磁磚計畫

本文件記錄了 AntiGravity 磁磚計畫外掛的開發歷程與重大修正內容。

## 🎯 V3.4 版本亮點 — UX 邏輯加固版

**版本 V3.4 (2026-03-14)** 掃除了介面交互中的「邏輯地雷」，大幅提升系統穩定性：

- ✨ **輸入防護網 (Input Hardening)** — 全面採用 `double.TryParse` 攔截格式錯誤，杜絕因格式錯誤導致的 DLL 崩潰。
- ✨ **全鍵盤支持 (Keyboard Accessibility)** — 完整綁定 `Enter` (確定) 與 `Esc` (取消)，無需滑鼠即可快速完成設定。
- ✨ **動態引導介面 (Dynamic Context)** — 優化交丁排面板的連動顯示邏輯，引導使用者專注於當前選定的排列式樣指標。

👉 **詳細技術文檔**: 見 [V34_TECHNICAL_GUIDE.md](V34_TECHNICAL_GUIDE.md)

## [V3.4.0] - 2026-03-14 (UX 邏輯加固)

### Fixed

- **地雷邏輯修復**：修正非數字輸入可能導致的 Parse Error。
- **UI 段落強化**：優化 `ValidateInputs` 檢核機制，區分「大於 0」與「不小於 0」的提示。
- **快速鍵綁定**：確保全視窗範圍內的 Enter/Esc 鍵盤映射。

## [V3.3.0] - 2026-03-14 (UI 交互優化)

### Added

- **雙向獨立對調**：新增 `BtnSwapSize_Click` 與 `BtnSwapGrout_Click`。
- **UI 佈局更新**：在磁磚尺寸與灰縫設定區塊分別加入對應的對調按鈕。

## [V3.2.0] - 2026-03-14 (幾何生命更新)

### Fixed

- **幾何生命週期修復**：修正拉伸宿主後需點擊兩次按鈕的 Bug。
- **強制環境刷新**：在抓取 Facet 幾何前強制執行 `Regenerate()`。

## [V3.1.0] - 2026-03-14 (單一交易架構)

### Fixed

- **單一主交易 (Master Transaction)**：廢除 `TransactionGroup`，成功掛載 `WarningSwallower`。
- **100% 靜默重繪**：徹底消滅 Revit 向上遞延產生的 HTML 錯誤報告彈窗。

### Changed

- **全視圖支援**：優化零件切換邏輯，完美支援 3D、平面、剖面視圖。

## [V3.0.0] - 2026-03-14 (UI 與 3D 支援)

### Added

- **UI 布局重組**：將顯示按鈕分開，提升操作效率。
- **3D 視圖與樣板支援**：精準判斷參數狀態，完美支援 3D 視角。

## [V2.5.0] - 2026-03-14 (核心架構優化)

### Added

- **核心架構優化**：導入母體溯源機制，解決子零件碎片化問題。
- **邊界保護機制**：引入 2mm 容差過濾，解決外邊緣退縮問題。

## [V2.0.0] - 2026-03-12 (零件架構核心)

### Added

- **零件切割架構**：從傳統族群架構轉換為 Revit Parts 切割架構。
- **隱形標註鎖定**：實作移動格線即同步更新排磚邏輯。
